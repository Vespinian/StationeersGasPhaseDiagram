# Stationeers Gas Phase Diagram

An interactive visualization of saturation curves for gases in the game [Stationeers](https://store.steampowered.com/app/544550/Stationeers/).

Originally inspired by [VerisZG's Stationeers Gas Phase Diagram](https://gralewski.github.io/stationeers-gas-phase-diagram-by-zgralewski.htm).

## Features

- Interactive pan and zoom on the phase diagram
- Toggle individual gases on/off
- Linear or logarithmic pressure scale
- Light and dark mode
- Hover tooltips showing pressure values at any temperature

## Developing

```sh
bun run dev
```

## Building

```sh
bun run build
```

You can preview the production build with `bun run preview`.
